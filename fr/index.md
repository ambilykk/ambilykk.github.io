## À propos des produits GitHub ##

GitHub propose des produits gratuits et payants pour stocker et collaborer sur du code. Certains produits s’appliquent uniquement aux comptes personnels, tandis que d’autres plans s’appliquent uniquement aux comptes d’organisation et d’entreprise. Pour plus d’informations sur les comptes, consultez « Types de comptes GitHub ».

Vous pouvez voir les prix et une liste complète des fonctionnalités pour chaque produit sur https://github.com/pricing. Pour plus d’informations sur les fonctionnalités et les produits prévus, consultez la feuille de route publique GitHub.
